Wat is Python

Wat is python? Python is een krachtige programmeertaal die makkelijk te leren is en waar je van alles mee kunt maken. Ook voor beginnende developers. Steeds vaker hoor of lees je over de taal, en dat is niet voor niets. Python wordt bijvoorbeeld veel gebruikt bij de ontwikkeling van webapplicaties en voor data science. Je zult er verbaasd over staan wat jij in korte tijd allemaal met Python kan.
